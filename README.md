## About The Project

Light and basic definition of catppuccin colors for latex.
Please see [Cappuccin for LaTeX](https://github.com/catppuccin/latex) for much more complete package.

In here the color are define following the [Cattpuccin palette](https://catppuccin.com/palette/) without prefix.

<!-- GETTING STARTED -->
## Getting Started

Use the package option `latte`, `frappe`, `machiatto` or `mocha` following your preferences:

```latex
\usepackage[latte]{latexppuccin}
```

Then you can use direcly the defined colors:

```latex
\textcolor{rosewater}{LaTeXppuccin}
```
